Overview


This game was created over 3 months of work, it was for my third project in my games dev college course. This game came out a success in my eyes, I will continue updating it to fix arising bugs, but for now, it is a great project that I worked on. I created all the 3D models, all the textures, blueprints and all the voice lines by myself. This game has ~5 minutes of playtime where you need to complete puzzles to gain access to the next room.
