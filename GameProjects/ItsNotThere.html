It's Not There

This is a game I created for the final project in my games development course in college

Home Games 3D Misc About Me
Blueprints Blender Photoshop Substance Painter

This game was made with Blender and UE5. The textures and art were modified and created in Photoshop and Krita. Substance Painter was also used. In this game you need to do tasks that make the overseer happier, if you do the wrong thing he will dislike you, what emotion will you get? This is a short game ~10 min playtime.

Overview


This game was created over 3 months of work, it was for the first project in the second year of college. This game came out a success in my eyes, I will continue updating it to fix arising bugs, but for now, it is a great project that I worked on. I created all the 3D models, all the textures, blueprints and all the voice lines by myself. This project was great but was a bit much to handle by myself, but overall I loved how it turned out, and I hope you do too.
