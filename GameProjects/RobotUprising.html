UE5 - Blueprints Blender Photoshop

I created this project in my own free time, I loved learning new things throughout this project.

This small game is a wave-based game where the number of agents increases every few rounds and the game becomes progressively difficult.

Overview


I created this game with learning new things in mind. I wanted to learn about post-processing materials, and how to add a weapon to the player by making a simple dynamic weapon system. So I created a wave-based game where you have to fend off robots from killing you, in the art style of a pixel game. This game was made for a little game jam in the second year of my college course. The theme was upgrade, so in this game you will kill the robots and upgrade your gun as you do each wave.
